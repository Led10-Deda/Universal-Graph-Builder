How to write the function

Here you can build any kind of function graph, as long as you write the function based on some rules:

  • Always use the variable x and write in lowercase type.
  • Use the following operators: + , - , * , / .
  • To make specific functions like trigonometric, logarithmic or exponential ones, use the given keyboard.
  • Use this syntax for:
    1. Power-->
      Math.pow(a,b) where a is the base and b is the exponent.
    2. Logarithm --> Math.log(a)/Math.log(b) where b is the base and a is the expression.
    3. Sine, Cosine or Tangent-->
      Math.sin(a) where a is the expression.
